Análisis de complejidad

1. Insertion
 Complejidad es de O(n)

2. Search
Complejidad es de O(n)

3. Update
Complejidad es de O(n)

4. DeleteAt
Complejidad es de O(n)

todos los metodos tienen una complejidad de O(n) por que es necesario recorrer toda la "Double Linked List" para efectuar las modificaciones deseadas.

Sin embargo se podría optimizar el tiempo de compilación si dividieramos la lista en dos y dependiendo de si está más cerca de tail o de head para no tener que recorrer toda la lista.
